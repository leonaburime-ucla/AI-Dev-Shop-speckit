# Maintainers

This folder holds maintainer-only material that should not be treated as active runtime context for normal feature delivery.

- `guides/`: implementation notes, historical design docs, and migration writeups
- `skill-md-format/`: rollout framework, overlays, and rewrite drafts for instruction-format work

Keep runtime rules in `AGENTS.md`, `agents/`, `skills/`, `workflows/`, and `project-knowledge/`. Use this folder for framework evolution work.
